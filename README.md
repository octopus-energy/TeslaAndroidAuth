# TeslaAndroidAuth
Lightweight Android library for Tesla SSO Authentication based on [Tesla JSON API (Unofficial)](https://tesla-api.timdorr.com)

## Setup (WIP)

- `mavenCentral` artifact coming soon

## Usage

```
TeslAuth(
        onError = {
            // Handle error
        },
        onSuccess = {
            // Handle success
        },
        loadingIndicator = {
            // Handle loading
        }
    )
```
